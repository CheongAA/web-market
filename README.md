## 기간

* 2020.07.22 ~ 진행중

## Tech/framework 

* Language: JDK 1.8
* Application Framework: Spring MVC 4.3.9
* Database: MySQL 8.0.16 + MyBatis 3.4.1
* Deployement: AWS EC2 , AWS RDS
* Storage: AWS S3
* Front-End: JSP , Javascript
* IDE: Eclipse, JRebel
* Build Automation Tool: Maven
